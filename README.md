# StreamFlix DevOps Project

StreamFlix is a StreamFlix-style streaming frontend application used as a DevOps portfolio project.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Firebase
- TMDB API

## DevOps Goals
- Dockerize the application
- Add Jenkins CI/CD pipeline
- Integrate SonarQube
- Deploy with Nginx
- Push image to Docker Hub
